Xolus

Leave a star on github
  • JavaScript
  • TypeScript
  • Nodejs

Xolus is an ongoing project aimed at developing a framework for creating web applications as a solo developer or small team. This project aims at creating applications interactable at first sight without waiting on any hydration techniques and to simpify the development of HTML and JavaScript applications by taking out the complexity of "known" frameworks.

Mainly, it serves more as a JSX templating tool for generating HTML on the server to be served to the browser. It also includes ways to pre-generate HTML files to be stored and served as static files. It comes with a developer-friendly environment for compiling the JSX code to JavaScript code runnable on the server.
